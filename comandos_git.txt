git init . - cria um repo git na pasta atual
git status - mostra o status da area de trabalho
git add <nome_arquivo> - add arquivo para ser comitado
git commit -m "msg" - faz o commit
git log - mostra todo os commits


## FLUXO DE TRABALHO
1. criar o diretório (pasta) do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "Mensagem estranha"
8. git status
9. volta para o passo 3
